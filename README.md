### 📌 Featured Projects

**[RAG_streamlit](https://github.com/dev-ilman/RAG_streamlit)**
A Retrieval-Augmented Generation app for querying PDF, DOCX, TXT, and CSV documents. Uses Google Gemini for the LLM, FAISS for vector search, and LangChain for orchestration, with a Streamlit UI and login/auth support. Includes OCR (Tesseract) for scanned PDFs.

**[MOSIP OCR Field Extraction & Verification](https://github.com/Simhaatt/ocr)**
An end-to-end document OCR and identity verification pipeline. Extracts fields from ID documents (Aadhaar, DL, passport, birth certificate) with PaddleOCR, maps them by document type, and fuzzy-verifies them against applicant-submitted data. FastAPI backend with a stubbed MOSIP integration layer.

**[Money Muling Detection](https://github.com/Simhaatt/money_muling_detection)**
A graph-based fraud detection platform that flags money mule networks in transaction data. Builds a transaction graph with NetworkX, extracts features (PageRank, betweenness centrality, cycle detection, Louvain communities), and runs a 10-stage risk-scoring pipeline with an interactive React/Cytoscape.js dashboard.
